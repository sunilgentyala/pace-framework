# Glossary of Key Terms and Acronyms

| Acronym / Term | Full Form | Definition |
|---|---|---|
| ACD | Active Cyber Defense | Defensive cyber actions that affect adversary infrastructure |
| ANH | Automated Network Healing | PACE Pillar I autonomous defensive action protocol |
| CIKR | Critical Infrastructure and Key Resources | Civilian infrastructure systems operationally relevant to military readiness |
| CPPE | Cyber-Physical Protection Element | Proposed new CPT sub-element for ICS/OT defense |
| CPT | Cyber Protection Team | USCYBERCOM team assigned to defend priority missions |
| CISA | Cybersecurity and Infrastructure Security Agency | DHS agency responsible for civilian CI security |
| CNMF | Cyber National Mission Force | USCYBERCOM component responsible for national-level cyber operations |
| DCIP | Defense Critical Infrastructure Program | DoD program identifying defense-critical infrastructure |
| DSCA | Defense Support of Civil Authorities | Legal authority for military assistance to civilian entities |
| EADM | Escalation-Aware Deterrence Matrix | PACE Pillar IV deterrence communication classification tool |
| EDR | Endpoint Detection and Response | Network security monitoring technology |
| ICS | Industrial Control System | Systems controlling physical infrastructure (power, water, etc.) |
| JP 3-12 | Joint Publication 3-12 | DoD doctrine for Cyberspace Operations |
| LLM | Large Language Model | AI model for processing and generating language at scale |
| MDI | Mission Dependency Index | PACE scoring metric for civilian infrastructure military significance |
| MRT-C | Mission Relevant Terrain in Cyberspace | Doctrinal concept identifying cyber terrain relevant to military ops |
| NCF | National Critical Functions | CISA framework categorizing critical national functions |
| NDAA | National Defense Authorization Act | Annual U.S. defense policy and authorization legislation |
| NRMC | National Risk Management Center | CISA center for CI risk management |
| NSC | National Security Council | Presidential advisory body for national security |
| OT | Operational Technology | Hardware/software controlling physical processes |
| PRC | People's Republic of China | Primary strategic competitor referenced in framework threat model |
| ROE | Rules of Engagement | Legal and operational parameters governing military actions |
| SAAP | Secure AI Analytics Pipeline | PACE Pillar II air-gapped LLM intelligence fusion architecture |
| USCYBERCOM | U.S. Cyber Command | DoD unified combatant command for cyberspace operations |
